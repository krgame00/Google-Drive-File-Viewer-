Google Drive File Viewer

วางลิงก์ Google Drive แล้วสำรวจ เข้าโฟลเดอร์ ดูรูป เล่นวิดีโอได้เลย

⚙️ ตั้งค่า API & บัญชี — ตั้งค่าไว้แล้ว ✓ (คลิกเพื่อแก้ไข)
ยังไม่ได้ตั้ง Client ID
ยังไม่ได้ตั้ง API Key
ยังไม่ได้ลงชื่อเข้าใช้ Google
📖 คู่มือตั้งค่า (คลิกเพื่อขยาย)
ขั้นที่ 1: เปิด Google Cloud Console → สร้าง Project
ขั้นที่ 2: เมนู APIs & Services → Library → ค้นและ Enable "Google Drive API"
ขั้นที่ 3 (API Key): APIs & Services → Credentials → Create Credentials → API key → คัดลอกมาวางในช่อง API Key ด้านบน
ขั้นที่ 4 (OAuth): OAuth consent screen → เลือก External → กรอกชื่อแอป + อีเมล → บันทึก
ขั้นที่ 5: Credentials → Create Credentials → OAuth client ID → ประเภท Web application
ขั้นที่ 6: ในส่วน Authorized JavaScript origins เพิ่ม URL ของหน้านี้ (เช่น http://localhost หรือที่เปิดไฟล์)
ขั้นที่ 7: คัดลอก Client ID มาวางในช่องด้านบน → กด บันทึก → กดปุ่ม "ลงชื่อเข้าใช้ Google"
รูปแบบลิงก์ไม่ถูกต้อง โปรดตรวจสอบอีกครั้ง

ยังไม่ได้เปิดโฟลเดอร์

วางลิงก์ Google Drive ข้างบนแล้วกด "เปิด"
